## How to add a project

A project needs:
1. A clear **outcome** — what does "done" look like?
2. A **next action** — what's the very next physical step?
3. A **deadline or target date** (even a rough one)

If it has no deadline and no finish line → it's an [[Areas/00 - Areas Overview|Area]], not a project.
