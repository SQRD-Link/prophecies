# 🗂️ PARA Index

> PARA is a four-category system for organising everything you know and do.
> Created by Tiago Forte — [Building a Second Brain](https://www.buildingasecondbrain.com/)

---

## 📁 Structure

| Folder | What goes here | Has a deadline? |
|---|---|---|
| [[Projects]] | Active goals with a finish line | Yes |
| [[Areas]] | Ongoing responsibilities to maintain | No |
| [[Resources]] | Topics you're interested in | No |
| [[Archive]] | Completed, paused, or inactive items | — |

## 💡 How to use

1. **Capture** → Everything lands in `docs/inbox.md` first
2. **Process** → Weekly: move inbox items into the right PARA folder
3. **Distil** → Summarise each note in your own words, don't just paste links
4. **Express** → Use your notes to create output (blog posts, projects, decisions)

> The goal is not to collect more notes. The goal is to *use* them.
